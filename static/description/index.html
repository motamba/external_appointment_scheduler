External Appointment Scheduler

Seamless Calendar Integration for Customer Appointments

Key Features

Customer-Facing Portal
Beautiful, responsive booking interface that integrates seamlessly with your Odoo portal. Customers can view real-time availability, select time slots, and book appointments instantly.

Google Calendar Integration
Full OAuth2-based integration with Google Calendar. Two-way synchronization ensures appointments are always in sync between Odoo and external calendars.

Service Management
Define multiple appointment services with custom durations, pricing, buffer times, and booking rules. Set minimum lead times, maximum advance booking windows, and cancellation policies per service.

Automated Notifications
Send automatic confirmation emails when appointments are booked, reminder emails before appointments, and cancellation notifications. Keep your customers informed every step of the way.

Advanced Capabilities

  • Multiple Calendar Views: List, Calendar, Kanban, and Form views for comprehensive appointment management
  • Webhook Support: Real-time updates via webhooks for instant synchronization
  • Appointment Status Tracking: Draft, Confirmed, Checked-In, Completed, Cancelled, No-Show
  • Customer Portal Access: Customers can view, cancel, and reschedule their appointments
  • Flexible Booking Rules: Set capacity limits, buffer times, and availability windows
  • OAuth2 Security: Secure authentication with Google Calendar API

Configuration & Settings

Easy Setup
Connect your Google Calendar in just a few clicks. The module guides you through the OAuth2 authorization process and automatically configures webhook notifications for real-time updates.

Granular Control
Configure default booking rules, reminder schedules, cancellation policies, and notification preferences from a centralized settings interface. Apply global defaults or customize per service.
